# InLinkr Branding Guide

## Platform vs Product

### Platform
**InLinkr** - The platform name for the link tracking and attribution system.

### Product
**TubeLinkr** - The YouTube-specific product built on the InLinkr platform.

### Shared Object
**Smart Link** - A trackable link that can be shared across multiple placements.

## Official Vocabulary

| Term | Usage |

## Terminology Rules

### Change to InLinkr (Platform-level)
- Dashboard titles and headings
- Account references
- General platform features
- Navigation (where applicable)
- Empty states
- Button text for link actions
- Marketing copy about the platform
- Support text about the platform

### Keep as TubeLinkr (Product-specific)
- YouTube-related features
- Placements
- Traffic Proof
- Video Performance
- Connect YouTube
- YouTube attribution language

### Keep Unchanged (Implementation details)
- API field names
- Database column names
- Environment variable names
- Wrangler configuration
- Worker configuration
- Database schema
- Database queries
- Test files
- Legacy documentation

## URL Examples

### Development
- Smart Link base: `go-dev.inlinkr.com`

### Production (TubeLinkr - unchanged)
- Smart Link base: `go.tubelinkr.com`
- Branded subdomain: `username.tubelinkr.com`

## Empty State Guidelines

**Avoid:** "No TubeLinks yet."
**Use:** "You haven't created any links yet."

**Avoid:** "Your TubeLinkr dashboard is empty."
**Use:** "Your dashboard is ready. Create your first link to get started."
